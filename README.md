## 🌟 Características Principales

Este proyecto no es solo una interfaz; es una aplicación funcional que consume datos reales y ofrece una experiencia de usuario premium.

- **🎧 Reproducción Real**: Integración con iTunes Search API para escuchar previews de 30 segundos de canciones reales.
- **❤️ Sistema de Favoritos**: Guarda tus canciones preferidas con persistencia local (`localStorage`).
- **📱 Totalmente Responsive**: Adaptado para móviles, tablets y escritorio con un menú lateral colapsable.
- **🎨 Diseño UI Premium**: Estética moderna con gradientes dinámicos, modo oscuro/claro y micro-animaciones.
- **⚡ Navegación Fluida**: SPA (Single Page Application) ultra rápida gracias a React y Vite.

## 🛠️ Tecnologías y Arquitectura

### Frontend Stack
- **React 18**: Biblioteca principal para la interfaz de usuario.
- **TypeScript**: Tipado estático para un código robusto y mantenible.
- **Tailwind CSS**: Estilizado moderno y eficiente.
- **Zustand**: Gestión de estado global ligera y potente para el reproductor de audio.
- **React Router Dom**: Manejo de rutas y navegación entre páginas.
- **Lucide React**: Set de iconos elegantes y minimalistas.

### Arquitectura
El proyecto sigue una estructura limpia y modular:
- `/src/components`: Componentes reutilizables (Card, Player, Sidebar, Header).
- `/src/pages`: Vistas principales (Home, Search, Library, Playlist, Favorites).
- `/src/store`: Estado global del reproductor manejado con Zustand.
- `/src/services`: Lógica de consumo de la API de iTunes.

## 🚀 Instalación y Uso

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/rogercipa7/spotify-clone.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Ejecutar en modo desarrollo:
   ```bash
   npm run dev
   ```
4. Construir para producción:
   ```bash
   npm run build
   ```
